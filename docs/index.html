waboxWhatsApp ↔ your filesystem

Incoming messages land in an inbox folder; drop a JSON in outbox to reply. No API to learn — just files. Built on Baileys.

Get started
Integration guide
View on GitHub
📥

Inbox — incoming

Every WhatsApp message (text + media) becomes a JSON file plus its attachment, dropped into your inbox folder.

📤

Outbox — outgoing

Write a small JSON job to send text, files, quote-replies and emoji reactions. wabox watches and sends.

🤖

Agent-ready

Point any process at the two folders, or install the skill — npx skills add rodgco/wabox — and your agent knows the contract.

🖥️

Runs as a service

One interactive setup installs a background service (systemd / launchd / Task Scheduler) that starts on login.
